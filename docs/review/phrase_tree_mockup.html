Tkinter Treeview (カテゴリ別分類表示) イメージモックアップ

以下は、定型文タブを「カテゴリ階層を持つツリー構造」に変更した際のTkinter画面のシミュレーションです。
カテゴリ行（グレー背景）をクリックすると、そのカテゴリに所属する定型文を展開・折りたたみできます。

定型文 / パターン (PhraseListComponent)
検索パターン (定型文) パターン / 正規表現
▼Programming (Python)
Python: Function ^def\s+\w+\(
Python: Class ^class\s+\w+[:\(]
Import statements ^import\s+|^from\s+
▼Structure / Strings
JSON Key \"w+\"\s*:
UUID / GUID [0-9a-fA-F]{8}-[0-9a-fA-F]{4}...
▼Network / Web
IPv4 Address \b(?:(?:25[0-5]|...
URL (HTTP/HTTPS) https?://[\w/:%#\$...
※実際のTkinterの ttk.Treeview でも、これと同様に折りたたみ可能な親子構造（インデント付き）を表現します。
